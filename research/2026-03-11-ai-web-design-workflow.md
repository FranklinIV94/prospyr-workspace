# Gemini 3.1 Pro is INSANE for Web Design

> **Subtitle:** Free AI UI Generator Workflow
> **Author:** Viktor Oddo (@viktoroddy)

---

## Overview

A complete workflow for building stunning, animated websites using AI tools. The author demonstrates creating a space-flora themed landing page with liquid glass effects and cosmic animations.

**Tools Used:**
- Pinterest (inspiration)
- Nano Banana Pro 2 (image generation)
- Kling 3.0 (video/animation)
- Mux (video hosting)
- Lovable (AI web builder)
- Motionsize.ai (component templates)

---

## The 5-Step Workflow

### Step 1: Find the Inspiration

**Process:**
1. Scroll Pinterest for "hero sections" with desired vibe
2. Find ethereal space + floral theme
3. Zoom in, crop to background only, screenshot
4. Upload to Nano Banana Pro 2
5. Use exact prompt:

> *"give me the background of it, exact background, no need for any other text or buttons and the bottom on logos, same color, creating of the background as in shot by in higher quality."*

**Result:** Clean high-resolution background

---

### Step 2: Build the Hero Section

**Process:**
1. Animate with Kling 3.0
2. Use exact prompt:

> *"fixed camera, locked lens, no camera movements, ultra slow cinematic motion, a very gentle breeze moves through the scene."*

3. Get 6-second seamless loop
4. Upload to Mux for hosting
5. Grab liquid glass UI from Motionsize.ai (free)
6. Use Lovable to recreate, swap in video background

**Result:** Animated hero with liquid glass effects

---

### Step 3: Generate the Full Landing Page

**Full Prompt:**

> *"build out the rest of the landing page. So I'm building a web design AI agency landing page. So rewrite all of the text on here to fit that, but keep the same number of characters as well as build the rest of the landing page. So I assume we would have a feature section, a benefit section with a chess kind of style where we have the content on the left and then the picture on the right and then on the bottom section we have the section where the content is on the right and the picture is on the left. So just two sections like that then have some feature section which would be a four cards section then let's have a testimonial section and maybe we would have some stats and any other sections that you think would fit about five to six sections. Build that in the same style that we already created in the hero section. So the same colors, the same fonts, the same styles for the buttons, for the cards, keep the slick wood glass, keep the same sizes for the fonts, et cetera, and keep the background pure black. Do not add any gradients, do not add any animation to the background, just keep it pure, pure black."*

**Result:** Full page in just 2 prompts:
- Hero + chessboard sections
- 4 glass cards
- Stats section
- CTA

---

### Step 4: Build the Rest of the Pages

**Process:**
- Copy hero prompt style
- Prompt: *"build out the about us page… use the same style. Five sections on About Us page."*

**Result:** Matching About Us page with same glass styling

---

### Step 5: Post on Social Media

**Process:**
1. Screen record the hero or full page
2. Write caption like:

> *"Liquid glass on the website hit differently Built this entire space-flora landing page with Kling 3.0 + Lovable in 15 mins"*

3. Tag relevant companies or @viktoroddy

## Key Insight

> *Without these detailed prompts you get generic AI garbage. With them? You ship stuff that looks hand-crafted.*

The difference between generic output and professional results is **specific, detailed prompts**.
