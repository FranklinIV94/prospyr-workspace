# Seed Skill: Context Engineering

Describes how to inject and structure context for agents to reason effectively.

## Summary
Guides the agent on when and how to fetch, prune, and layer context.

## Prompts & Patterns
- Essential-context: list of must-have facts for decision
- Noise-filter: rules to prune irrelevant data
- Layered-context: stack privacy, security, and domain hints
