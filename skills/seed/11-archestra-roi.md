# Seed ROI: Archestra

Objective: Evaluate enterprise orchestration with cost-aware routing for multi-provider setups.

## Value proposition
- Reduces cross-provider latency and simplifies governance across OpenRouter/Anthropic/OpenAI.
- Improves cost visibility through centralized routing decisions.

## Key metrics to watch
- Latency reduction (ms)
- Token-cost reduction per task
- Governance events logged

## Risks & mitigations
- Potential integration drift: start with a narrow scope (one routing path) and validate before broader rollouts.
- Security: ensure access controls for routing decisions; audit trails preserved.
