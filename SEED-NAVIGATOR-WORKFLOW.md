# Seed Navigator Workflow (Draft)

This workflow demonstrates how to traverse the seed graph from knowledge-work into seed notes, and how to validate seeds in sandbox.

## Steps
1. Start at knowledge-work (seed graph traversal map).
2. Open Seed Graph Index.
3. Visit seed-discovery, seed-context-engineering, seed-memory-management.
4. From memory seed, navigate to ClawKeep and Archestra seeds.
5. Validate links and run a dry-run import into memory seeds.
6. Record results and decisions (ROI, risk, next steps).

## Validation
- Ensure wikilinks resolve (no broken links)
- No code execution from seed notes
- Document diff for any new seeds
